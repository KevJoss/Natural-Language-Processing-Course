# Natural-Language-Processing-Course

This repository contains practical exercises, laboratories, and projects developed for the Natural Language Processing (NLP) class at Yachay Tech University.

## Repository Structure

Below is a brief description of the main directories:

* **`NLP_getting_started/`**: Introductory folder. Contains the first steps in text analysis, focusing on Exploratory Data Analysis (EDA) and preprocessing of a disaster tweets dataset (from Kaggle).
* **`Laboratories/`**: Space intended to host the code for regular practices and laboratories to be completed throughout the course.
