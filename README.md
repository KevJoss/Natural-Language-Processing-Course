## 📂 Repository Structure

Below is a brief description of the main directories in this repository:

### 🚀 `NLP_getting_started/`
Introductory folder for the course. It covers the first steps in text analysis with a Kaggle **"Disaster Tweets"** dataset. 
* **Key contents include:**
  * 📊 Comprehensive Exploratory Data Analysis (EDA)
  * 🧹 Data cleaning and text preprocessing
  * 🤖 Introductory Deep Learning and Neural Network models for text classification

### 🧪 `Laboratories/`
Space intended to host the code for regular practices, assignments, and laboratories to be completed throughout the entire course.
